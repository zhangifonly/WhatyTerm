## 实验设计要素

### 研究假设

```markdown
无显著差异/无相关性

存在显著差异/存在相关性
```

### 变量定义
| 变量类型 | 说明 | 示例 |
|----------|------|------|
| 自变量 | 研究者操控的变量 | 药物剂量 |
| 因变量 | 测量的结果变量 | 治疗效果 |
| 控制变量 | 保持恒定的变量 | 年龄、性别 |

## 样本量计算

### 功效分析
```python
from statsmodels.stats.power import TTestIndPower

analysis = TTestIndPower()
sample_size = analysis.solve_power(
    effect_size=0.5,  # 效应量
    alpha=0.05,       # 显著性水平
    power=0.8         # 统计功效
)
```
