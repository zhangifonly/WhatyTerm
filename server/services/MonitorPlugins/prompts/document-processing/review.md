# 文档处理 - 审核校对阶段

## 阶段目标
检查文档内容和格式的正确性。

## 审核内容

### 内容检查
- 信息是否准确
- 数据是否正确
- 逻辑是否通顺
- 是否有遗漏

### 格式检查
- 字体是否统一
- 间距是否一致
- 对齐是否正确
- 样式是否规范

### 语言检查
- 拼写是否正确
- 语法是否正确
- 标点是否正确
- 术语是否一致

## 校对工具

### 拼写检查
- Word 内置
- Grammarly
- LanguageTool

### 格式检查
- Word 样式检查
- LaTeX 编译警告

## 校对流程

```
1. 自动检查（拼写、语法）
2. 格式检查
3. 内容审核
4. 最终确认
```

## 审核清单

- [ ] 内容是否有错误
- [ ] 格式是否正确
- [ ] 是否符合要求
