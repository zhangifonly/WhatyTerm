设计页面布局和交互流程。

### 布局模式
| 模式 | 适用场景 |
|------|----------|
| 单栏 | 文章、博客 |
| 双栏 | 后台管理 |
| 三栏 | 邮件客户端 |
| 卡片 | 商品展示 |
| 瀑布流 | 图片展示 |

## 交互设计

### 用户流程
```
首页 → 列表页 → 详情页 → 操作 → 结果
```

### 交互状态
- 默认状态
- 悬停状态
- 点击状态
- 禁用状态
- 加载状态
- 错误状态
