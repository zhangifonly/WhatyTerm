# 论文写作 - 文献研究阶段

## 阶段目标
系统收集和整理相关文献，建立理论基础。

## 文献检索策略

### 1. 确定检索关键词
```
主题词 + 同义词 + 相关词
例如：
- 主题：机器学习
- 同义词：machine learning, ML
- 相关词：深度学习, 神经网络, AI
```

### 2. 选择数据库
| 数据库 | 适用领域 | 特点 |
|-------|---------|------|
| Google Scholar | 综合 | 覆盖广，免费 |
| Web of Science | 综合 | 权威，引用分析 |
| PubMed | 医学生物 | 专业，免费 |
| IEEE Xplore | 工程技术 | 专业 |
| CNKI | 中文文献 | 国内权威 |

### 3. 检索技巧
```
# 布尔运算符
"machine learning" AND "healthcare"
"deep learning" OR "neural network"
"AI" NOT "artificial insemination"

# 通配符
comput* (computer, computing, computation)

# 精确匹配
"natural language processing"

# 限定条件
- 时间范围：近5年
- 文献类型：期刊论文
- 语言：英文
```

## 文献管理

### 文献记录模板
```markdown
## 文献信息
- 标题：
- 作者：
- 期刊/会议：
- 年份：
- DOI：

## 核心内容
### 研究问题

### 研究方法

### 主要发现

### 研究贡献

## 与本研究的关系
- 相关性：高/中/低
- 可引用点：
- 研究空白：

## 引用格式
[APA/MLA/Chicago 格式]
```

### 文献分类
| 类别 | 说明 | 用途 |
|------|------|------|
| 核心文献 | 直接相关 | 重点引用 |
| 支撑文献 | 理论基础 | 背景介绍 |
| 方法文献 | 研究方法 | 方法论证 |
| 对比文献 | 类似研究 | 讨论对比 |

## 文献综述写作

### 综述结构
1. **主题组织**：按主题分类讨论
2. **时间组织**：按发展历程讨论
3. **方法组织**：按研究方法分类

### 综述要点
- 不是简单罗列，要有分析
- 指出研究趋势和空白
- 建立与本研究的联系

## 检查清单

- [ ] 是否覆盖了主要相关文献
- [ ] 是否包含近期研究（近3-5年）
- [ ] 是否有经典/奠基性文献
- [ ] 文献记录是否完整
- [ ] 是否识别了研究空白
