# 论文写作 - 内容撰写阶段

## 阶段目标
按照大纲撰写论文各章节内容。

## 写作原则

### 1. 学术写作规范
- 使用正式、客观的语言
- 避免第一人称（或按期刊要求）
- 使用专业术语
- 保持逻辑清晰

### 2. 段落结构
```
主题句（Topic Sentence）
↓
支撑句（Supporting Sentences）
↓
过渡句（Transition Sentence）
```

### 3. 引用规范
```markdown
# 直接引用
According to Smith (2020), "exact quote here" (p. 15).

# 间接引用
Previous research has shown that... (Smith, 2020).

# 多作者引用
(Smith & Jones, 2020)
(Smith et al., 2020) - 3人以上
```

## 各章节写作要点

### 摘要
- 独立成文，不引用
- 包含：背景、目的、方法、结果、结论
- 字数：200-300词

### 引言
```markdown
## 写作顺序
1. 大背景 → 小背景（漏斗式）
2. 问题陈述
3. 研究目的
4. 研究意义
5. 论文结构预览
```

### 文献综述
- 按主题/时间/方法组织
- 批判性分析，不是简单罗列
- 指出研究空白
- 建立与本研究的联系

### 方法
- 详细到可复现
- 说明选择理由
- 承认局限性

### 结果
- 客观呈现数据
- 使用图表辅助
- 不做过多解释（留给讨论）

### 讨论
- 解释结果含义
- 与已有研究对比
- 讨论理论和实践意义
- 承认局限性

### 结论
- 总结主要发现
- 强调贡献
- 提出未来方向

## 写作技巧

### 提高可读性
- 使用短句和简单句
- 一段一个主题
- 使用过渡词连接

### 常用过渡词
| 关系 | 过渡词 |
|------|-------|
| 递进 | Furthermore, Moreover, In addition |
| 对比 | However, In contrast, On the other hand |
| 因果 | Therefore, Thus, As a result |
| 举例 | For example, For instance, Specifically |
| 总结 | In conclusion, To summarize, Overall |

## 检查清单

- [ ] 是否按大纲结构写作
- [ ] 段落是否有主题句
- [ ] 引用是否规范
- [ ] 逻辑是否连贯
- [ ] 语言是否学术化
