# 论文写作 - 大纲规划阶段

## 阶段目标
制定清晰的论文结构和写作大纲，为后续写作提供框架。

## 论文结构模板

### 学术论文标准结构
```markdown
# 论文标题

## 摘要 (Abstract)
- 研究背景（1-2句）
- 研究问题/目的（1句）
- 方法概述（1-2句）
- 主要发现（2-3句）
- 结论/意义（1句）
- 关键词（3-5个）

## 1. 引言 (Introduction)
### 1.1 研究背景
### 1.2 问题陈述
### 1.3 研究目的
### 1.4 研究意义
### 1.5 论文结构

## 2. 文献综述 (Literature Review)
### 2.1 理论基础
### 2.2 相关研究
### 2.3 研究空白

## 3. 研究方法 (Methodology)
### 3.1 研究设计
### 3.2 数据收集
### 3.3 数据分析
### 3.4 研究局限

## 4. 结果 (Results)
### 4.1 描述性统计
### 4.2 主要发现
### 4.3 假设检验

## 5. 讨论 (Discussion)
### 5.1 结果解释
### 5.2 与已有研究对比
### 5.3 理论贡献
### 5.4 实践意义

## 6. 结论 (Conclusion)
### 6.1 主要发现总结
### 6.2 研究贡献
### 6.3 研究局限
### 6.4 未来研究方向

## 参考文献 (References)

## 附录 (Appendix)
```

## 大纲编写要点

### 1. 明确研究问题
- 研究什么问题？
- 为什么这个问题重要？
- 现有研究的不足是什么？

### 2. 确定论证逻辑
- 主要论点是什么？
- 如何支撑论点？
- 论证的逻辑顺序？

### 3. 规划各章节内容
| 章节 | 内容要点 | 预计字数 |
|------|---------|---------|
| 摘要 | 全文概述 | 200-300 |
| 引言 | 背景、问题、目的 | 1000-1500 |
| 文献综述 | 理论、研究现状 | 2000-3000 |
| 方法 | 研究设计、数据 | 1500-2000 |
| 结果 | 数据分析结果 | 1500-2000 |
| 讨论 | 解释、贡献 | 1500-2000 |
| 结论 | 总结、展望 | 500-800 |

## 检查清单

- [ ] 研究问题是否明确
- [ ] 论文结构是否完整
- [ ] 各章节逻辑是否连贯
- [ ] 是否有明确的论证主线
- [ ] 字数分配是否合理
