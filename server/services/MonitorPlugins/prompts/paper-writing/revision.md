# 论文写作 - 修改润色阶段

## 阶段目标
对初稿进行修改和润色，提升论文质量。

## 修改层次

### 1. 结构层面
- 论证逻辑是否清晰
- 章节安排是否合理
- 段落过渡是否自然

### 2. 内容层面
- 论点是否充分支撑
- 证据是否充足
- 是否有遗漏或冗余

### 3. 语言层面
- 表达是否准确
- 语法是否正确
- 风格是否一致

## 常见问题

### 结构问题
| 问题 | 解决方案 |
|------|---------|
| 逻辑跳跃 | 添加过渡段落 |
| 重复冗余 | 合并或删除 |
| 详略不当 | 调整篇幅 |

### 语言问题
| 问题 | 示例 | 修改 |
|------|------|------|
| 口语化 | "很多" | "大量" |
| 模糊 | "一些研究" | "Smith (2020) 的研究" |
| 冗长 | "由于...的原因" | "由于..." |

## 润色技巧

### 提升学术性
```
# 避免
This paper will talk about...
I think that...

# 使用
This paper examines...
The findings suggest that...
```

### 增强逻辑性
```
# 使用信号词
First, Second, Finally
However, Nevertheless
Therefore, Consequently
```
