# 通用监控 - 空闲状态

## 状态说明
当前终端处于空闲状态，可能的情况：

1. **任务已完成** - 上一个任务执行完毕，等待新指令
2. **等待继续** - Claude Code 完成了一个步骤，等待用户确认继续
3. **会话中断** - 之前的操作被中断，需要恢复

## 判断标准

### Claude Code 空闲
- 显示 `>` 提示符
- 没有错误信息
- 没有正在执行的任务

**操作**: 发送 `继续`

### Shell 命令行空闲
- 显示 `$` 或 `#` 提示符
- Claude Code 可能已退出

**操作**:
- 如果需要继续开发，发送 `claude -c` 重新启动
- 如果任务已完成，无需操作

### Python REPL 空闲
- 显示 `>>>` 或 `In [n]:` 提示符

**操作**: 根据任务需要输入 Python 代码或退出

## 注意事项

- 不要在程序运行中发送指令
- 确认当前状态确实是空闲而非等待输出
- 如果不确定，等待几秒观察是否有新输出
