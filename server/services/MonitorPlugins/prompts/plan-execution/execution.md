# 计划执行 - 逐步执行阶段

## 阶段目标
按计划执行任务，跟踪进度。

## 执行原则

### 专注当前
- 一次只做一件事
- 完成后再开始下一个
- 避免多任务切换

### 及时记录
- 记录执行过程
- 记录遇到的问题
- 记录解决方案

## 执行流程

```
1. 选择当前任务
2. 理解任务要求
3. 执行任务
4. 验证结果
5. 标记完成
6. 选择下一个任务
```

## 进度跟踪

### 任务状态
| 状态 | 说明 |
|------|------|
| 待开始 | 尚未开始 |
| 进行中 | 正在执行 |
| 已完成 | 执行完成 |
| 已阻塞 | 遇到问题 |

### 进度更新
```markdown
## 今日进度

### 已完成
- [x] 任务 1

### 进行中
- [ ] 任务 2（50%）

### 待开始
- [ ] 任务 3
```

## 问题处理

### 遇到阻塞
1. 记录问题
2. 分析原因
3. 寻求帮助
4. 调整计划

## 执行清单

- [ ] 当前任务是否完成
- [ ] 是否遇到阻碍
- [ ] 进度是否正常
