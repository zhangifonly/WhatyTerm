# 计划执行 - 头脑风暴阶段

## 阶段目标
发散思考，收集各种想法和方案。

## 头脑风暴原则

### 基本规则
1. **不批评** - 暂时不评判想法
2. **自由发挥** - 鼓励大胆想法
3. **数量优先** - 先求多再求精
4. **组合改进** - 在他人想法上发展

### 思维方法
- 发散思维：尽可能多的想法
- 收敛思维：筛选最佳方案
- 横向思维：跳出常规思考

## 头脑风暴流程

```
1. 明确主题
2. 自由发想（10-15分钟）
3. 记录所有想法
4. 分类整理
5. 评估筛选
6. 确定方向
```

## 记录模板

```markdown
## 头脑风暴记录

### 主题
如何提高用户留存率？

### 想法列表

2. 积分奖励系统

4. 个性化推荐

### 分类
#### 产品功能
- 积分系统
- 社交功能

#### 运营策略
- 推送通知
- 活动促销

### 优先方案
1. 积分奖励系统
2. 个性化推荐
```
