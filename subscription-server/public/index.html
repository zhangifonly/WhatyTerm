常见问题

个人用户真的完全免费吗？

是的。WhatyTerm 已开源，所有功能（包括全部监控策略和 AI 分析）对个人用户完全免费，支持最多 3 台设备。

企业版有什么额外服务？

企业版提供专属中转服务器（更稳定的远程连接）、不限设备数、专属技术支持和定制功能开发。

如何获取企业版？

前往 用户中心 注册账户，选择企业版计划并完成支付即可。

源代码在哪里？

WhatyTerm 在 GitHub 上开源，欢迎 Star 和贡献代码。
